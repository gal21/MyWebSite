My Answers

Answer 1:

Web 1:
In Web 1, there is only limited interaction between sites and web users. Static pages and web sites, Data updated once in a while.
Web 1 is simply an information portal where users passively receive information without being given the opportunity to post reviews, comments, and feedback.
Examples : personal web pages, and Info pages.

Web 2:
Unlike Web 1, Web 2 facilitates interaction between web users and sites, so it allows users to interact more freely with each other.
Web 2 encourages participation, collaboration, and information sharing, Sites that allow users to contribute content, Sites that let users share content.
Examples of Web 2 applications are Youtube, Wiki, Flickr, Facebook, and so on.

Web 3:
Web 3 Its about making the internet more Indexed, With unified Semantics, Like one virtual DB e.g. read all possible posts in medicine and create a medical DB of question-answer.
In Web 3, computers can interpret information like humans and intelligently generate and distribute useful content tailored to the needs of users.
One example of Web 3 is Tivo, a digital video recorder. Its recording program can search the web and read what it finds to you based on your preferences.
